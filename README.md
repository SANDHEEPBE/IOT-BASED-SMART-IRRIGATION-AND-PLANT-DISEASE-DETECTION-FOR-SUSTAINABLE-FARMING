This project, IoT-Based Smart Irrigation and Plant Disease Detection for Sustainable Farming, integrates sensor-driven automation and cloud-based analytics to optimize water usage and detect crop diseases in real time, enhancing agricultural efficiency and sustainability; it leverages IoT devices for soil moisture monitoring, weather tracking, and image-based disease identification, with data synced to cloud platforms for remote access and decision-making, aiming to empower farmers with intelligent tools that reduce resource waste, improve crop health, and support scalable, eco-friendly farming practices.
